# Pilhas (Stack) - Listas

- The **last** element to **enter**, is the **first** to **leave**.
- **LIFO** (**L**ast **I**n, **F**irst **O**ut). Examples: Browser history, `CTRL + Z`
- Stack Underflow: Pop in a empty list.
- Stack Overflow: Push in a full list.
